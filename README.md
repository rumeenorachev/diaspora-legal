# Plataforma SaaS para academias de idiomas

Este proyecto consiste en el desarrollo de una plataforma SaaS diseñada para digitalizar la gestión de academias de idiomas.

La plataforma permite administrar estudiantes, cursos, profesores y evaluaciones a través de una aplicación móvil conectada a un backend centralizado.

Actualmente el sistema se encuentra en fase piloto y será probado inicialmente por la academia **Diaspora Languages Zaragoza**.

---

# Arquitectura del sistema

La plataforma está compuesta por dos partes principales:

### Aplicación móvil
Desarrollada con:

- Flutter
- Dart

### Backend
Desarrollado con:

- Django
- Python
- API REST

### Base de datos

- PostgreSQL

---

# Funcionalidades principales

La plataforma incluye funcionalidades como:

- Gestión de estudiantes
- Gestión de profesores
- Gestión de cursos
- Sistema de exámenes y calificaciones
- Comunicación interna entre academia y estudiantes
- Aplicación móvil para acceso a la información académica

# 🏫 Plataforma SaaS para academias de idiomas

Este repositorio contiene la documentación pública y el contenido legal de una plataforma SaaS que he desarrollado para la gestión de academias de idiomas.

El sistema permite digitalizar la gestión académica mediante una aplicación móvil conectada a un backend centralizado.

Actualmente la plataforma se encuentra en fase piloto y será utilizada inicialmente por la academia **Diaspora Languages Zaragoza**.

---

# 🧱 Arquitectura del sistema

La plataforma está compuesta por dos partes principales:

## 📱 Aplicación móvil

Desarrollada con:

- Flutter
- Dart

## ⚙️ Backend

Desarrollado con:

- Django
- Python
- API REST

## 🗄️ Base de datos

- PostgreSQL

---

# 🚀 Funcionalidades principales

La plataforma incluye funcionalidades como:

- Gestión de estudiantes
- Gestión de profesores
- Gestión de cursos
- Sistema de exámenes y calificaciones
- Comunicación entre academia y estudiantes
- Aplicación móvil para acceso a la información académica
